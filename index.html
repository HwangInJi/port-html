01

Team Project

첫번째 팀 프로젝트로 진행했던
마라톤 정보 공유 사이트 포트폴리오 입니다.

마라톤 정보 공유 사이트

EXPLANATION:

코로나 이후 마라톤에 대한 관심이 높아지면서 현재는 마라톤을 즐기는 사람도, 마라톤 대회도 이전에 비해 많아지게 되었습니다. 이 사이트는 그 중에서도 2030을 타겟으로 하여 젊은 마라토너들이 마라톤 정보나 일정을 확인하고, 이용자들끼리 다양한 지식과 정보를 공유할 수 있는 커뮤니티 사이트를 목적으로 만들게 되었습니다.

  • CODE VIEW →

티겟이 2030인만큼 MZ세대에게 힙하고 현대적인 분위기와 이미지의 사이트로 다가가기 위해 직접 우리만의 캐릭터를 만들어 디자인을 진행했고, 전체적인 컬러를 따뜻함과 열정을 상징하는 오렌지 컬러를 사용해 러너들 사이에서 친근하고 활기찬 분위기를 조성 및 러닝의 즐거움과 창의성을 강조하는 웹사이트로 만들고자 했습니다.

커뮤니티(게시판)페이지는 자유게시판, 크루모집, 번개모임, 질문게시판 등 각 게시판마다 링크를 연결시켜 원하는 페이지를 사용할 수 있도록 개별적인 PHP를 만들어 코드를 연결시켰으며, 직관적으로 보일 수 있도록 UI/UX 디자인을 했습니다.

02

Team Project

두번째 팀 프로젝트로 진행했던
티켓 판매처별 순위 정보 사이트 포트폴리오 입니다.

티켓처 순위 정보 사이트

EXPLANATION:

멜론, 예스24, 티켓링크에서 진행중인 뮤지컬, 전시, 콘서트의 순위 정보를 볼 수 있는 사이트입니다. 매일 자동 수집된 데이터가 사이트에 연결되어 있기 때문에 작품의 순위 변동 및 새로 순위권에 진입하게 된 작품들을 한 사이트에서 일자별, 카테고리별로 한번에 확인할 수 있다는 장점을 갖고 있습니다.

  • SITE VIEW →
  • CODE VIEW →

포스터 이미지가 하나의 액자처럼 보일 수 있게 카드 형태로 디자인 및 한 화면에 4개씩 보여질 수 있도록 틀을 잡고 해당 포스터에 마우스 커서를 올리면 3D효과처럼 돌아가며 순위, 제목, 진행 기간, 장소 정보를 보여주도록 만들었습니다.

Select옵션을 통해 각 사이트별 콘서트, 전시, 뮤지컬을 선택할 수 있고, 오른쪽 Calendar에서 일자를 선택하면 해당 일자의 JSON 데이터가 자동으로 불러와지도록 하였고, 추가로 포스터 이미지를 누르면 팝업 형태가 뜨면서 순위의 변동 유무까지 포함된 정보창과 함께 예매하기 링크를 통해 사이트로 이동되도록 설정하여 순위와 예매까지 동시에 한 곳에서 진행할 수 있도록 만들었습니다.

03

Team Project

세번째 팀 프로젝트로 진행했던
잡플래닛같은 전국 네일샵 익명 리뷰 사이트 포트폴리오 입니다.

전국 네일샵 리뷰 사이트

EXPLANATION:

우리 일상 속에서 네일샵은 이제 흔하게 볼 수 있는 공간이 되었습니다. 그러면서 자연스레 네일샵의 증가 또한 많아지게 되었고, 선택의 폭도 넓어지면서 자연스럽게 이곳이 정말 괜찮은 곳인가? 하는 정보 또한 필요해졌습니다. ‘오늘네일’은 그런 네일샵 정보를 알고 싶어하는 이용자들을 위한 목적으로 만든 사이트입니다.

  • SITE VIEW →
  • CODE VIEW →

대부분의 이용고객이 여성인 네일샵이기 때문에 포인트 컬러는 핑크색 계열로 잡았고, 아이콘같은 여성들이 좋아하는 꾸미기 요소를 넣어 메인 디자인을 잡았습니다. 또한 상단에는 별점, 리뷰수가 많은 네일샵을 랭킹으로 하여 볼 수 있도록 하여 어디를 가야 하는지 모를 이용자들의 고민을 메인에서 조금이나마 해결할 수 있도록 레이아웃을 잡았습니다.

전국 네일샵 데이터 중 네이버플레이스에 등록된 샵을 기준으로 하여 기본적인 정보를 가져와 이용자들이 상세정보를 통해 볼 수 있도록 하였고, 오른쪽에는 리뷰창을 넣어 이용자들의 익명 리뷰를 볼 수 있습니다.

정보처리기능사 필기 시험문제 퀴즈 사이트 포트폴리오 입니다.

정보처리기능사 필기 퀴즈

EXPLANATION:

정보처리기능사 필기 시험을 준비하면서 만든 사이트 입니다. 시험을 준비하는 과정에서 어렵거나, 헷갈렸던 문제들을 모아서 주관식으로 만들었고 직접 풀어보면서 틀리면 해설을 확인하고, 나중엔 몇 문제를 맞췄는지도 확인 할 수 있도록 사이트를 제작하였습니다.

  • SITE VIEW →
  • CODE VIEW →

메인페이지는 퀴즈 게임같은 느낌을 주기 위해 배경으로 움직이는 캐릭터들 소스를 스크립트 기능을 활용하여 넣었고, 시작하기 버튼을 누르면 주관식 시험을 볼 수 있습니다.

문제에 대한 답을 입력했을 때 틀리면 설명이 나오고, 정답일 경우엔 설명이 뜨지 않고 다음 문제로 넘어가도록 기능을 구현하였습니다. 위의 내가 몇 번째 문제를 풀고 있는지 확인할 수 있고, 메인 화면을 누르면 문제를 풀다가도 다시 처음으로 돌아갈 수 있습니다. 마지막엔 총 문제 수에서 내가 몇 문제를 맞췄는 지 확인이 가능합니다.

자바스크립트&MySQL

EXPLANATION:

처음으로 자바스크립트를 배울 때 수업시간에 배운 기초 지식들을 정리한 사이트입니다. 데이터를 어떻게 불러오고, 수식을 쓸 때 필요한 구문이 무엇인지를 예시와 함께 정리했고, MySQL에선 테이블을 어떻게 만드는 지, 그 테이블 안에 데이터를 어떻게 넣는지 등을 예시와 함께 정리하여 필요할 때마다 보면서 복습할 수 있게 정리해두었습니다.

  • SITE VIEW →
  • CODE VIEW →

자바스크립트에서 말하는 객체, 배열의 기초부터 이를 활용한 데이터 실행 방법, 데이터 실행 시 사용되는 구문에 대해 정리해둔 사이트입니다.

MySQL의 기본 정의부터 설치 및 셋팅 방법, 데이터 베이스를 만들고 테이블을 만들어 그 안에 데이터값을 넣고 삭제하는 방법 등 기초적인 SQL에 대해 정리해두었습니다.

유튜브 API를 활용하여 만든 사이트 포트폴리오 입니다.

Tea 정보 검색 사이트

EXPLANATION:

‘Taste Tea’는 차 마시는 것을 좋아해서 이를 주제로 하여 만들어본 사이트입니다. 유튜브 API를 활용해 만들었기 때문에 Tea와 관련된 모든 유튜브 영상을 가져올 수 있고 재생은 물론 해당 영상의 댓글, 유튜버 채널까지 확인할 수 있습니다. 추가적으로 Tea정보 외에 모든 검색이 가능하여 유튜브처럼 이용할 수 있습니다.

  • SITE VIEW →
  • CODE VIEW →

메인페이지 중앙에는 귀여운 배너 이미지를 활용해 디자인을 넣었고, 하단엔 Tea와 관련된 영상 하나가 새로고침 할 때마다 랜덤으로 재생되며, 검색어를 입력하면 해당 관련 영상들을 볼 수 있도록 기능을 구현하였습니다.

메뉴탭에 있는 카테고리를 누르면 해당 키워드와 연관된 유튜브 영상이 50개가 나오며, 하단에 더보기 버튼을 누르면 추가로 50개씩 나오도록 설정하여 해당 키워드 영상을 모두 볼 수 있도록 기능을 구현했습니다. 추가로 해당 채널의 정보도 볼 수 있습니다.

07

Solo Project

개인적으로 만든 프로젝트로 음원 사이트별 차트 순위와
나만의 플레이리스트를 만들 수 있는 사이트 포트폴리오 입니다.

음악 차트&플리 사이트

EXPLANATION:

‘Music Chart’는 대형 음원 사이트인 멜론, 벅스, 지니, 애플, 빌보드 차트의 100위까지의 순위를 알 수 있고 유튜브 API를 활용해 해당 노래에 대한 정보 및 영상 재생은 물론 나만의 폴더를 만들어 플레이 리스트를 완성할 수 있는 사이트입니다. 오른쪽 플레이 리스트 영역에서 노래를 재생하면 해당 노래의 유튜브 영상까지 볼 수 있습니다.

  • SITE VIEW →
  • CODE VIEW →

메인 페이지에선 다른 사람들의 리스트 JSON파일 데이터 곡들을 가져오고, 별을 눌러서 오른쪽 플레이 리스트에 추가, 삭제할 수 있도록 하여 폴더를 만들지 않고도 메인페이지에서 플레이리스트를 만들 수 있도록 기능을 구현하였습니다.

음원 사이트에서 불러오는 각각의 순위 데이터를 누르면 해당 노래와 연관된 유튜브 영상이 5개 보여지고, Search를 통해 검색하면 50개의 관련 영상을 볼 수 있습니다. 사이트에서 불러오는 모든 데이터는 다 플레이 리스트에 추가가 가능하도록 기능을 설정해두었습니다.

08

Solo Project

개인적으로 만든 프로젝트로
영화에 대한 정보를 찾아볼 수 있는 사이트 포트폴리오 입니다.

영화 정보 검색 사이트

EXPLANATION:

‘Sky Watch’는 TMDB에서 제공하는 영화 데이터 소스를 활용하여 만든 사이트입니다. 영화 정보에 대해 한 사이트에서 모두 해결할 수 있도록 만든 영화 사이트이기 때문에 사용자는 이곳에서 신작, 개봉예정작, 평점높은영화를 메인에서 확인할 수 있고, 각 각의 영화에 대한 상세 정보 및 티저 영상을 볼 수 있습니다.

  • SITE VIEW →
  • CODE VIEW →

메인 페이지는 Aside 구조로 레이아웃을 만들었으며, 상단엔 검색 기능과 커뮤니티, 로그인이 있고 신작, 개봉예정, 평점높은영화 순으로 영화 정보를 버튼을 누르면 스크롤 방식으로 넘기며 볼 수 있습니다. 여기에 포스터에 마우스 커서를 올리면 상세정보와 티저영상을 볼 수 있도록 기능을 구현했습니다.

상세정보 버튼을 누르면 영화에 대한 기본 정보, 티저영상을 볼 수 있고 출연진, 추천영화는 자동 슬라이드 방식으로 구현하여 따로 누르지 않아도 볼 수 있도록 기능을 구현하였습니다.
